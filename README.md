# Nota & Todo Studio

Een luxe, soepele notitie- en todo-app met automatische opslag.

## Starten

Open `index.html` direct in je browser of start een simpele webserver:

```bash
python -m http.server 8000
```

Ga vervolgens naar `http://localhost:8000/index.html`.

## GitHub-weergave

GitHub toont standaard alleen de README. Om de app te zien:

- **Lokaal openen:** download of clone de repo en open `index.html` (of gebruik de server hierboven).
- **GitHub Pages:** activeer Pages in de repo-instellingen en kies de root (`/`) als bron. Daarna kun je de app via de Pages-URL openen.
